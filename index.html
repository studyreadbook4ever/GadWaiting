Ad slot test surface

GadWaiting, officially 개드웨이팅, is a tiny TypeScript library for the waiting period before a web ad program is approved or reliably serving ads.

This static demo runs without a local server. It imports dist/gadwaiting.js, swaps between mock provider states, and keeps the slot dimensions fixed.

Modes

  • Fallback: uses a missing provider script, so the local creative appears.
  • Network: uses a mock script that fills the ad slot.
  • Unfilled: uses a mock script that reports an empty ad unit, then falls back.

Drop-in usage

<div id="ad-sidebar"></div>
<script type="module">
  import { mountGadWaiting } from "./dist/gadwaiting.js";
  mountGadWaiting(config);
</script>
