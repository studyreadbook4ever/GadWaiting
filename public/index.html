Ad slot test surface

GadWaiting, officially 개드웨이팅, is a tiny TypeScript library for the waiting period before a web ad program is approved or reliably serving ads.

When the ad provider script is unavailable, times out, or reports an unfilled slot, GadWaiting shows your own local image ad in the same space.

What this demo proves

  • Fallback mode shows the local GadWaiting... creative.
  • Network mode renders a mock provider creative in the same fixed slot.
  • Unfilled mode falls back after the provider reports an empty ad unit.

Quick start

npm run doctor
npm run demo

Drop-in usage

<div id="ad-sidebar"></div>
<script type="module">
  import { mountGadWaiting } from "/dist/gadwaiting.js";

  const config = await fetch("/gadwaiting.config.json").then((response) => response.json());
  mountGadWaiting(config);
</script>
